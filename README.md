# SimpleBack
悬浮辅助按钮

* 因为不习惯现有辅助按键程序的操作方式，所以写了此程序
* 单次点按为返回，长按为改变按钮位置
* 使用滑动操作来选择所需其他功能（截图，返回桌面，锁屏）
* 因android接口限制，截图功能只适用于5.0以上版本
* 使用前需先开启辅助权限，以及程序的悬浮窗显示权
